# Фронт для чата
## Готовое приложение
Работающее приложение можно посмотреть по [ссылке](https://quiet-mesa-02189.herokuapp.com/messenger/)
## Сборка
Сборка локально осуществляется с помощью Webpack

`npm run dev`

Либо так

`npm run build`

## Запуск
Приложение запускается на 3000 порту командой

`npm start`
